Data Protection Policy

Satofin takes data protection and privacy very seriously. We implement appropriate technical and organizational measures to protect user data.

Data Collection

We only collect user data necessary to provide Satofin services. This may include name, email, address, transaction details, etc. We collect the minimum data needed.

Data Storage

Any personal data is stored securely in encrypted form. Access is restricted only to necessary personnel. Data is protected from loss or destruction.

Data Usage

User data is only used for providing Satofin's services. Data is not shared or sold externally without explicit consent.

Data Transfer

Satofin only transfers data across borders in compliance with data protection laws. Appropriate safeguards are used for any data transfers.

Data Rights

Users can request access to, correction of, or deletion of their stored personal data at any time. Users can object to certain uses of data.

Breach Notification

Any personal data breaches will be notified to affected users and regulators within 72 hours as required by law.

Contact us if you have any questions about how Satofin protects user data and privacy.
